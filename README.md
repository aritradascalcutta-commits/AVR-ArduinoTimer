# AVR Timer with TM1637 Display and Buzzer

## Project Overview
This project is a countdown timer built using an AVR microcontroller.
The timer output is shown on a TM1637 4-digit display.
When the timer reaches zero, an active buzzer beeps.

## Components Used
- AVR Microcontroller
- TM1637 4-digit display
- Active buzzer
- Jumper wires

## How It Works
The microcontroller controls the display and updates time.
When countdown ends, buzzer is triggered.
